# UI/UX trendy 2025/2026 — rešerše pro kolo 2 (W3.3)

Datum: 2026-07-07 · Účel: podklad pro směry 4–7 (`docs/design/mockups/smer-4..7.html`).
Metoda: web search, orientační (ne hloubková studie). U každého trendu poznámka,
jak se dá využít pro Habitnaut.

## Co je teď skutečně živé

1. **Apple Liquid Glass (iOS 26)** — nový systémový jazyk Apple ohlášený na WWDC
   červen 2025, nasazený napříč iOS/macOS/watchOS 26: translucentní „skleněný"
   materiál s refrakcí a hloubkou, plovoucí ovládací prvky, vícevrstvé ikony,
   adaptivní světlo/tma. HIG důraz: text vždy na plné vrstvě (ne přímo na skle),
   kontrast min. 4.5:1 po bluru. Third-party appky jazyk přebírají.
   → Habitnaut: směr 4 — „first-party iOS" dojem pro web PWA.

2. **Bento grids** — z Apple-marketingového vzoru se stal standard; verze 2026 má
   výraznější zaoblení rohů a taktilní mikrointerakce uvnitř dlaždic. Půlroční
   „reality check" články potvrzují, že bento drží.
   → Habitnaut: směr 5 — den jako modulární paluba dlaždic.

3. **Dark-first + calm/quiet UI** — dark mode už není trend, ale očekávaná
   preference (OLED, oči večer); profesionální týmy designují dark-first a light
   odvozují. Souběžně „calm design": tlumené palety (Pantone 2026 = Cloud Dancer),
   kognitivní klid, žádná vizuální divadla, mikrointerakce jen jako potvrzení.
   → Habitnaut: směr 6 — večerní 2min check-in je přesně calm/dark use case.

4. **Anti-grid / neo-brutalismus a „schematická syrovost"** — protipohyb k bento:
   raw, schématické, brutálně čitelné layouty; „intentional incompleteness";
   data bez dekorace; mono písma, viditelná struktura. Označováno jako jeden
   z nečekaných příběhů 2026.
   → Habitnaut: směr 7 — check-in jako letový záznam, data mluví sama.

5. **Expresivní/kinetická typografie** — oversized titulky, variabilní fonty,
   typografie jako hlavní nositel značky. Reality check: drží jako „polish",
   ne jako celá identita — používat v dávkách.
   → Napříč směry: velké titulky data/čísel (streak, den).

6. **Ustálené vzory (ne novinky, ale očekávání):** bottom sheets jako standardní
   kontejner sekundárního obsahu, gesture-first navigace s okamžitou odezvou,
   strategický glassmorphism (jen pro dočasné vrstvy), přístupnost jako výchozí
   požadavek, AI-adaptivní personalizace (pro nás pokrývá kouč, ne UI vrstva).
