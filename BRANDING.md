# Habitnaut

Rozhodnuto Michalem 2026-07-06 (viz docs/research/2026-07-W3-name-candidates.md, kolo 1).

- **Název:** Habitnaut — habit + astronaut. Sourozenec Michalova „Kafkanauta".
- **Výslovnost:** CZ „habitnaut", EN /ˈhæbɪtnɔːt/. Funguje v obou jazycích.
- **Kolize:** žádná nalezená appka/doména/repo (orientační web search 2026-07-06; před registrací
  domény udělat WHOIS). Fonetická blízkost HabitNu — jiná kategorie, nízké riziko.
- **Příběh:** průzkumník návyků. Každý den = jeden oblet (orbit), check-in = zápis do lodního
  deníku (mission log — sedí ke git commitům!), streak = stabilní orbita, vynechání = korekce
  kurzu (ne havárie — kontinuita > výkon, „nikdy 2× po sobě").
- **Tón:** hravý průzkumník, ne korporát ani drill sergeant. Kouč je „řídicí středisko" — parťák,
  který hlídá trajektorii a jemně korieguje. Česky primárně, anglicky připraveno.
- **Tagline kandidáti:** „Tvoje denní orbita." · „Mission log pro tvoje návyky." ·
  EN: "Your daily orbit." / "A mission log for your habits."
- **Vizuální motivy pro design (W3.3):** orbity/prstence, checkpoint markery, lodní deník,
  hvězdná mapa historie (heatmapa!), decentně — žádný kýčovitý vesmír na pozadí.
- **Maskot:** malý astronaut — Michal vygeneruje přes nano banana (low prio, neblokuje).
- Přejmenování GitHub repa/Supabase projektu: až po ověření názvu používáním (viz W3 plán).
